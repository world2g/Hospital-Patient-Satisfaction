# Hospital Service Performance & Patient Satisfaction Analysis

This project is a comprehensive Data Analytics study aimed at understanding the key factors that influence **patient satisfaction** in a hospital setting. Using both exploratory and visual analytics, the project uncovers trends, operational challenges, and insights that can help improve hospital service delivery.


## Dataset Overview

The analysis uses the **Hospital Beds Management Database**, which contains multiple relational tables covering different aspects of hospital operations:

- Patients
- Weekly Services
- Staff
- Staff Work Schedule
